# CS Bank Case Study (Personal Project)

This repository contains a personal case study notebook demonstrating:
- loading and cleaning a messy CSV export,
- splitting data into normalized entities (Customers, Accounts, Branches, Employees, Transactions),
- a PK/FK logical model,
- product overview (counts + balances per currency),
- transaction analytics with fee inconsistency checks,
- 2023 profitability estimation using APR and transaction fees.

## How to run
Install requirements and open the notebook in `notebooks/`.

## Disclaimer
This repository is for demonstration purposes only. Data is either anonymized, synthetic, or provided for the case study.
